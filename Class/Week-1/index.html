A Beginner's Guide to Using GitHub

Getting Started

Atom is a text editor, you use it to write your code.

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

GitHub for desktop is an application downloaded onto your computer which allows you to track changes made to your code in your text editor and then easily commit and sync those changes to be viewed on your online repository.

Step by Step

  1. Download Atom, Create a Github account and download "Github for desktop".
  2. After you have all the necessairy software downloaded you will need to create a new repository within the Github website. This repository should be titled "yourusername.github.io"
  3. After you have downloaded the Github for desktop software you will need to sign in with your GitHub account.
  4. You are now going to need to clone the repository you created online to your computer. There are two ways to do this:
    1. The first way is to go to the repository that you created online, press the green button labeled "clone or download" Select open in desktop, the repository will then open in your desktop application.
    2. The second way is to open the Github Desktop software, press the "+" symbol in the upper left hand corner. Then tab to the clone option and select your desired repository from the list. You may be prompted to choose a save location, be sure to choose something that is easily accessable.
  5. After you have cloned your repository you will need to test whether or not everything is working properly. Open Atom and create a new index.html file. Save it in the location of your new cloned github repository. Create a simple paragraph tag and type "hello world". Save it and then reopen the github desktop software. The "changes" tab at the top should show that you have one uncommitted change. Clicking on this tab will allow you to view Your code.
  6. The next step is to commit your changes. You do this by adding a summary of what changes were made and pressing "Commit to Master"
  7. The final step is to press the sync button, Your changes will now be visible on the GitHub Website.
